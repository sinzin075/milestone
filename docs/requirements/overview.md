# アプリ概要

## 使用者層
- 達成したいことや継続したいことがある人

## 作成目的
- 達成したいことや継続したいことの最終目標（以降「ゴール」）と、途中でやるべきこと（以降「マイルストーン」）を明確化する
- ゴール／マイルストーンの完了状況を視覚化し、モチベーション維持につなげる

## 用語
- ゴール：最終的に達成したい目標
- マイルストーン：ゴール達成の途中で完了させるべきタスク（途中目標）

## 作成機能（MVP）
### ユーザー
- 登録
- ログイン
- ユーザー情報の更新
- メールアドレスの変更

### ゴール
- 登録
- 変更
- 削除
- 表示（一覧／詳細）

### マイルストーン
- 登録
- 変更
- 削除
- 表示（一覧／詳細）
- ゴールに対する進捗率の表示  
  - 進捗率 = 完了マイルストーン数 / 全マイルストーン数

## その他機能（将来対応）
- 通知機能
- カレンダー反映機能

## 作成しない機能
- 他ユーザー／SNSなどへの共有機能

## 扱うデータ（概要）
### ユーザー
- ユーザーID
- ユーザー名
- メールアドレス
- アイコン画像
- 誕生日
- パスワード（ハッシュ化して保存する）
- 作成日
- 最終更新日
- 退会日（論理削除）

### ゴール
- ゴールID
- ユーザーID（所有者）
- ゴール名称
- ゴールの概要説明
- ゴール完了予定日
- 作成日
- 最終更新日
- 削除日（論理削除）

### マイルストーン
- マイルストーンID
- ゴールID（紐づくゴール）
- タイトル
- 内容
- 予定日
- 完了フラグ（または完了日）
- 作成日
- 最終更新日
- 削除日（論理削除）

## 注意事項
- 本ドキュメントの内容は、作成進捗により変更される可能性がある
